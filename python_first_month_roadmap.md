# Python Learning Roadmap: First Month (Basics to Intermediate)

**Program Overview**: 2-month program  
- **Month 1**: Python Fundamentals (3 days/week, ~12 sessions)  
- **Month 2**: AI Automations (to be covered separately)

This roadmap takes a student from absolute beginner to confident intermediate Python programmer, with a strong foundation for automation and AI scripting.

## Overall Approach & Tips
- **Session Length**: 1.5–3 hours each (theory + coding + exercises).
- **Tools**: Python (latest version), VS Code (with Python extension) or PyCharm, Jupyter Notebook.
- **Resources**:
  - Free: Official Python Tutorial, "Automate the Boring Stuff with Python" (book/site), freeCodeCamp YouTube.
  - Practice: LeetCode (Easy), HackerRank, Codecademy.
- **Homework**: 30–60 minutes daily coding practice.
- **Assessment**: Weekly quizzes, mini-projects, and code reviews.
- **Teaching Style**: Explain "why" (real-world automation examples), live coding, and pair programming where possible.

---

## Week 1: Python Fundamentals (Days 1–3)

**Focus**: Syntax, basic logic, and interactive scripts.

### Day 1: Setup & Basics
- Python installation and running code (interpreter, `.py` scripts, Jupyter).
- Variables, data types (`int`, `float`, `str`, `bool`).
- Type conversion/casting.
- Operators (arithmetic, comparison, logical).
- f-strings and basic `print()` / `input()`.
- **Mini-project**: Simple calculator or personalized greeting script.

### Day 2: Control Flow
- Conditionals: `if`, `elif`, `else`.
- Loops: `for` and `while`, `break`/`continue`.
- Basic error handling with `try-except`.
- **Mini-project**: Number guessing game or simple quiz.

### Day 3: Strings & Basic File I/O
- String methods (`upper()`, `split()`, `join()`, `strip()`, formatting).
- Basic file reading/writing (text files).
- Practice problems combining loops and strings.
- **Homework**: Text-based menu system.

**Week 1 Goal**: Write simple interactive command-line scripts.

---

## Week 2: Core Data Structures & Functions (Days 4–6)

**Focus**: Organizing and manipulating data efficiently.

### Day 4: Lists & Tuples
- Lists: indexing, slicing, methods (`append`, `remove`, `sort`, `pop`).
- List comprehensions.
- Tuples and immutability.
- **Mini-project**: To-do list manager (CLI).

### Day 5: Dictionaries & Sets
- Dictionaries: creation, access, methods, iteration.
- Sets: operations (union, intersection, difference).
- **Mini-project**: Simple inventory system or contact book.

### Day 6: Functions
- Defining functions, parameters, return values.
- Scope, default/keyword arguments, `*args` & `**kwargs`.
- Lambda functions (introduction).
- **Mini-project**: Refactor previous projects using functions for modularity.

**Week 2 Goal**: Write clean, reusable, modular code.

---

## Week 3: Modules, Files & OOP Basics (Days 7–9)

**Focus**: Real-world scripting and code organization.

### Day 7: Modules & Packages
- Importing standard library modules (`math`, `random`, `datetime`, `os`, `sys`).
- Creating your own modules.
- Virtual environments (`venv`).

### Day 8: File Handling & Exceptions
- Advanced file operations (text, JSON, CSV basics with standard library).
- Robust exception handling.
- Context managers (`with` statement).
- **Mini-project**: File organizer or basic log analyzer.

### Day 9: Object-Oriented Programming (OOP) Introduction
- Classes, objects, attributes, methods.
- `__init__` constructor, inheritance basics.
- Encapsulation principles.
- **Mini-project**: Bank account simulator or student record system using classes.

**Week 3 Goal**: Build maintainable applications and understand Python's ecosystem.

---

## Week 4: Advanced Concepts & Capstone (Days 10–12)

**Focus**: Polish skills and prepare for automation.

### Day 10: Advanced Python Features
- Iterators and generators.
- Decorators (basic understanding).
- Regular expressions (`re` module) for text processing.
- Advanced comprehensions and functional programming concepts.

### Day 11: Scripting & Automation Tools
- Deep dive into `os`, `shutil`, `subprocess` for file/system automation.
- Working with JSON and basic data formats.
- Debugging techniques and best practices.
- **Mini-project**: Bulk file renamer or data processor script.

### Day 12: Review, Major Project & Next Steps
- Comprehensive review of all topics.
- Major capstone project: Personal automation tool (e.g., expense tracker, automated report generator, or backup script).
- Introduction to testing (`unittest` or `pytest` basics).
- Reflection: How these skills enable AI automations in Month 2.

**Week 4 Goal**: Complete a functional project and gain confidence for advanced topics.

---

## Recommended Projects
- **Beginner**: Games, calculators, quizzes.
- **Intermediate**: File handlers, inventory systems, class-based apps.
- **Capstone**: Real automation script solving a daily problem.

## Transition to Month 2 (AI Automation)
After completing Month 1, students will be ready for:
- Web scraping and APIs.
- Automation libraries (Selenium, etc.).
- Data processing for AI.
- Building simple AI-powered scripts (e.g., using OpenAI API).

Encourage students to maintain a GitHub repository for all projects.
